# 📚 Энциклопедия АвтоЭлектрик AI — Структура каталога

> **Контекст проекта:** Приложение специализируется на электрических системах автомобилей, AI-диагностике, чтении ошибок OBD и помощи автоэлектрикам. Энциклопедия должна быть практическим справочником.

---

## 🧩 Принципы построения дерева

| Принцип | Описание |
|---|---|
| **Специфика производителя** | Каждый бренд имеет уникальные блоки, протоколы и архитектуру |
| **Ориентация на электрику** | Приоритет — электросистемы, CAN-шины, ЭБУ, датчики |
| **Практическая ценность** | Каждый раздел содержит коды ошибок, схемы, методы проверки |
| **Уровни доступа** | Базовые статьи — для всех; схемы и процедуры — для специалистов |
| **Связь с диагностикой** | Статьи энциклопедии связаны с результатами AI-диагностики |
